user manual.

  1. Download NeutralizeIt from Releases and unzip the package.
  2. Using Google Chrome, go to Chrome Extensions (chrome://extensions).
  3. Enable "Developer mode" and click on "Load unpacked".
  4. Choose the root directory of NeutralizeIt.
  5. You will now be able to toggle the extension from the extensions page!
